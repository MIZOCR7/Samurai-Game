# Samurai Game 

A simple 2D fighting game built with Python and Pygame. It's a local 2-player game that supports both Keyboards and Joysticks/Controllers.

## Controls

### Player 1 (Ninja):
* **Move:** `A` / `D`
* **Jump:** `SPACE`
* **Attacks:** `Z`, `X`, `C`
* **Shield:** `Q`

### Player 2 (Samurai):
* **Move:** `H` / `K`
* **Jump:** `U`
* **Attacks:** `B`, `N`, `M`
* **Shield:** `Y`
